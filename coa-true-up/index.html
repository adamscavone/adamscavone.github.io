Handoff to David Kohn

Pull every CoA NCTL has issued out of Confident Cannabis and into NCTL's own archive.

North Coast Testing Laboratories generates Certificates of Analysis and currently depends on Confident Cannabis as the authoritative archive. We're closing that gap: every of-record CoA goes into NCTL's own Azure Files share, with cryptographic integrity and per-file provenance. After this ships, NCTL's internal LIMS becomes the system of record.

This project is independent of the in-flight Kondo branch of the NCLims codebase. It reads production read-only, talks to Confident's API, writes to the existing Azure Files share, and keeps its bookkeeping in its own isolated ledger. No NCLims source-code changes required. Pearson's microbials release is untouched.

Read all five sections below in order. They go from "what this project is" through "how the existing system works" to "the engineering plan, access requirements, and the API protocol." At the end, copy the bootstrap prompt and paste it into a fresh Claude Code session — that's how you start.
